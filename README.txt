Generates random Turkish student database with random names, lastnames, pob, dob, depts. Then transfers the json data to mysql.
İsimleri, soyisimleri, doğum yerleri, doğum tarihleri, numaraları ve bölümleri olan rastgele türk öğrenciler yaratır ve json olarak kaydeder. Sonrasında json dosyasını MySql serverine transfer eder.

JsonCreator: Creates students randomly. ( Rastgele isim, soyisim, doğum yeri ve departman seçerek tamamen rastgele bir veriseti yaratır )
JSONtoMySql: Transfers jsonData to mysql server
